## Pull Request

- 翻訳元サイトの変更の追従

- Jekyllテンプレートの改善
- ディレクトリ構成の改善

など、このリポジトリへの改善全般をPull Requestでいつでも受け付けています。

疑問点や質問はIssueを立てて相談をお願いします。
